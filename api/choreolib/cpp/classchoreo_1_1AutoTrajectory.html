Detailed Description

template<choreo::TrajectorySample SampleType, int Year = util::kDefaultYear>
class choreo::AutoTrajectory< SampleType, Year >

A class that represents a trajectory that can be used in an autonomous routine and have triggers based off of it.

Template Parameters
SampleTypeThe type of samples in the trajectory.
YearThe field year. Defaults to the current year.

Returns a trigger that has a rising edge when the command finishes, this edge will fall again the next cycle.

This is not a substitute for the Inactive() trigger, inactive will stay true until the trajectory is scheduled again and will also be true if thus trajectory has never been scheduled.
